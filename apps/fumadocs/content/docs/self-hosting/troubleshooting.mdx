## Quick Tips

**Port conflicts:**
```bash
sudo lsof -i :5499
sudo lsof -i :5498
```

**Check Redis:**
```bash
docker compose exec redis redis-cli -a $REDIS_PASSWORD ping
```

**View logs:**
```bash
docker compose logs -f
```
